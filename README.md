# Tic-tac-toe
This Tic-Tac-Toe project is a sleek,  built from scratch using vanilla HTML. CSS, and JavaScript. Designed with a focus on clean code and elegant aesthetics, it transforms a timeless favorite into a polished web application perfect for casual play.
It is a clean, responsive, and lightweight tic tac toe game built using vanilla HTML, CSS, and JavaScript. Featuring a modern dark mode UI, smooth animations, and automatic win/draw detection.



---

##  Features

*   Modern UI:Styled with a sleek, dark-themed color palette and vibrant accents for X and O markers.
*   Smooth Animations: Includes responsive hover effects and scale-down active click states.
*   **Smart Win/Draw Detection: Real-time checking for all 8 winning combinations and tie scenarios.
*   **Reset functionality:** A quick-response reset button to instantly clear the board and start a new round.
*   **Fully Responsive:** Scales dynamically to look great on desktop monitors, tablets, and smartphones.
